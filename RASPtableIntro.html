RASP viewer user guide

This page is derived from the work of Dr. Jack Glendening and Paul Scorer.

Selectors:

The Day, Time, Full/Reduced Parameter set and the Parameter itself are all selectable in the table on the left.

To copy the Page URL, Right-Click on “Page URL” and select “Copy Link Location” (the exact words differ between browsers). You can then paste this into an email, or whatever.

The Start Animation button can be used to advance the Time on a timer with the Delay time between steps. This works for the current selected Parameter. To stop, click the button again (will change into Stop Animation). The Delay is configurable using the spin box.

NB: do not set the Delay too short until all images for the current selected day have been loaded. After all RASP overlay images have been loaded, you can reduce the timer interval as short as 250 ms.

The Specify Task button can be used as a simple task planner. Turnpoints will be added as markers on the map. Placing the mouse on a turnpoint will give you the turnpoint name, Left Click to add it to the task and the distance so far will be shown. For a closed task, the Start-Finish point reports the End distance. Use Clear Task to remove the drawn task and Remove Waypoints to remove the turnpoint markers.

The AirSpace checkboxes show/hide restricted airspace overlays. You can select as many as you wish. NB: No NOTAMS and No Guarantees!

The Disclaimer button will show the legal disclaimer.

The Full Screen Toggle is a switch to enable full screen mode for browsers that support this (most do).

On the Map:

The Time can be advanced by a single Left-Click on the Map. It moves around circularly.

The Opacity of the Overlay (i.e. of the RASP maps) can be adjusted using the and buttons at Top-Right of the main map.

The background map type can be selected by using the layer control in the upper right corner. Currenlty we support 3 map types, “toner”, “terrain” and “OSM” (Open Street Maps).

The map can be zoomed with the ScrollWheel, by DoubleLeftClicking or by using the “+/-” buttons at bottom right (as normal with Maps). On devices with a touchscreen you can of course use the familiar two finger pinch and zoom gestures to zoom in or out.

The “Value” of a parameter can be displayed by Right-Clicking at a location. What you get depends on the selection radio buttons under Info PopUp on the left. “Day” provides values for all forecast times, not just the selected time of the map.

Note: when the value is displayed as a minus sign, then the data file for that parameter could not be found.

Soundings are available as a Parameter in the Full Set, selectable in the table on the left. They are also available by clicking the “S” symbols which gives you the Sounding forecast for that location and time. These soundings are quick to display on the map. If you wish to look in more detail, you can Right-Click and select “View Image” (you currently need the “Back” button to return).

Note that you may need to allow PopUps in your browser.

Note for Touch Screen Users

There is no Right-Click on a Touch Screen! A Right-Click is replaced by a “Long Click” (currently 3 seconds) - this works for a mouse too. Opacity is adjustable only with the buttons at Top-Right.
