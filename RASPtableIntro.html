Side-bar:

Use the "side-bar" menu to open the desired setting.

  • The "hamburger" menu will show the Selectors.
  • The "map" menu item will result in the simple task planner controls.
  • The "areas" menu item will result in a page where you can jump to another area or select the old GUI
  • The "heart" will show the support page that you can use show your love for this site :-).
  • The "eye" will open the IGC viewer.
  • The "?" will show the help controls.
  • The "talk" icon will show the disclaimer, and finally
  • The "link" item will show the URL control, further explained below.

Selectors

The Day, Time, Full/Reduced Parameter set and the Parameter itself are all selectable in the table on the left.

The Start Animation button can be used to advance the Time on a timer with the Delay time between steps. This works for the current selected Parameter. To stop, click the button again (will change into Stop Animation). The Delay is configurable using the spin box.

NB: do not set the Delay too short until all images for the current selected day have been loaded. After all RASP overlay images have been loaded, you can reduce the timer interval as short as 250 ms.

Simple Task Planner

The Specify Task button can be used as a simple task planner. Turnpoints will be added as markers on the map. Placing the mouse on a turnpoint will give you the turnpoint name, Left Click to add it to the task and the distance so far will be shown. For a closed task, the Start-Finish point reports the End distance. Use Clear Task to remove the drawn task and Remove Waypoints to remove the turnpoint markers.

IGC Viewer

This tool allows you to upload an IGC log file and have it displayed on the map. Nothing is uploaded to the server, the data is completely processed within your web browser.

Link Page

To copy the Page URL, Right-Click on “Page URL” and select “Copy Link Location” (the exact words differ between browsers). You can then paste this into an email, or whatever.

On the Map:

The Time can be advanced by a single Left-Click on the Map. It moves around circularly.

The Opacity of the Overlay (i.e. of the RASP maps) can be adjusted using the and buttons at Top-Right of the main map.

The background map type can be selected by using the layer control in the upper right corner. Currenlty we support 3 map types, “toner”, “terrain” and “OSM” (Open Street Maps). This control also allows to show/hide the Soundings and Meteograms markers, as well as the Airspace layer (courtesey of openaip.net).

The Full Screen Toggle is a switch to enable full screen mode for browsers that support this (most do).

The map can be zoomed with the ScrollWheel, by DoubleLeftClicking or by using the “+/-” buttons (as normal with Maps). On devices with a touchscreen you can of course use the familiar two finger pinch and zoom gestures to zoom in or out.

The “Value” of a parameter can be displayed by Right-Clicking at a location. What you get depends on the selection radio buttons under Info PopUp on the left. “Day” provides values for all forecast times, not just the selected time of the map.

Note: when the value is displayed as a minus sign, then the data file for that parameter could not be found.

Soundings are available as a Parameter in the Full Set, selectable in the table on the left. They are also available by clicking these marker symbols on the map, which gives you the Sounding forecast for that location and time. These soundings will open in a new window, so make sure to allow for popups for this site.

Meteograms are available as a Parameter in the Full Set, selectable in the table on the left. They are also available by clicking these marker symbols on the map, which gives you the Meteogram forecast for that location for the selected date. These meteograms will open in a new window, so make sure to allow for popups for this site. A brief explanation of a meteogram can be seen here.

Soundings/Meteograms are locations that offer both a Sounding and a Meteogram. These are identified on the map with the following symbol on the map, which gives you the Meteogram forecast for that location for the selected date when left clicked. Use Ctrl + Left Click to open the Sounding for the selected time. These meteogram or sounding plot will open in a new window, so make sure to allow for popups for this site.

Note for Touch Screen Users

There is no Right-Click on a Touch Screen! A Right-Click is replaced by a “Long Click” (currently 1 second) - this works for a mouse too. Opacity is adjustable only with the buttons at Top-Right.

Control + Left Click is also not available, as a work around use the Parameter selection in the Full Set.

No popup?

Don't want to see the popup when opening the website? Add "?nopopup" to the bookmark address.

For example: https://blipmaps.nl/NETHERLANDS/?nopopup

The same information (news) in the popup is also visible in the Help tab.
